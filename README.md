# ajibadeboluwatife.github.io

Personal portfolio site powered by GitHub Pages.

To run locally:
- Use any static file server, or:
  - python3 -m http.server 8000
  - then open http://localhost:8000

To publish:
- Create a repo named `ajibadeboluwatife.github.io`
- Push these files to the `main` branch
- Visit https://ajibadeboluwatife.github.io
